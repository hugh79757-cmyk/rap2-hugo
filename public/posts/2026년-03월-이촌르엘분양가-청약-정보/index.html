#

2026л…„ 03мӣ”м—җ м§„н–үлҗҳлҠ” көӯлҜјмһ„лҢҖмЈјнғқ мІӯм•Ҫ кіөкі  лӘ©лЎқмқҖ лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.

мІӯм•Ҫ мһҗкІ© л°Ҹ мӨҖл№„ м„ңлҘҳ
#

мІӯм•Ҫм—җ м§Җмӣҗн•ҳкё° мң„н•ҙм„ңлҠ” лӢӨмқҢмқҳ мһҗкІ© мҡ”кұҙмқ„ м¶©мЎұн•ҙм•ј н•©лӢҲлӢӨ.

мӢ мІӯ мһҗкІ© мҡ”кұҙ
#

  • л¬ҙмЈјнғқ м„ёлҢҖ: мІӯм•Ҫ мӢ мІӯмһҗлҠ” ліёмқё л°Ҹ м„ёлҢҖмӣҗмқҙ лӘЁл‘җ л¬ҙмЈјнғқмқҙм–ҙм•ј н•©лӢҲлӢӨ.
  • мҶҢл“қ мҡ”кұҙ: мҶҢл“қмқҙ мқјм • кё°мӨҖ мқҙн•ҳмқё кІҪмҡ°м—җ н•ңн•ҙ мӢ мІӯн•  мҲҳ мһҲмҠөлӢҲлӢӨ.
  • мһҗмӮ° мҡ”кұҙ: мһҗмӮ°мқҙ нҠ№м • кё°мӨҖ мқҙн•ҳмқҙм–ҙм•ј н•ҳл©°, мқҙлҘј мҙҲкіјн•  кІҪмҡ° мІӯм•Ҫмқҙ м ңн•ңлҗ©лӢҲлӢӨ.

мӨҖл№„ м„ңлҘҳ мІҙнҒ¬лҰ¬мҠӨнҠё
#

  • мЈјлҜјл“ұлЎқл“ұліё
  • мҶҢл“қ мҰқлӘ…м„ң (к·јлЎңмҶҢл“қмӣҗмІңм§•мҲҳмҳҒмҲҳмҰқ л“ұ)
  • мһҗмӮ° мҰқлӘ…м„ң (мҳҲ: мһ¬мӮ°м„ё лӮ©м„ё мҰқлӘ…м„ң)
  • мІӯм•ҪнҶөмһҘ мӮ¬ліё
  • кё°нғҖ кё°кҙҖм—җм„ң мҡ”кө¬н•ҳлҠ” м„ңлҘҳ

2026л…„ 03мӣ” мІӯл…„ мЈјнғқл“ңлҰј мІӯм•ҪнҶөмһҘ м •ліҙ

мІӯм•Ҫ мӢ мІӯ л°©лІ•
#

мІӯм•Ҫ мӢ мІӯмқҖ н•ңкөӯл¶ҖлҸҷмӮ°мӣҗ мІӯм•ҪнҷҲмқ„ нҶөн•ҙ м§„н–үлҗ©лӢҲлӢӨ. м ‘мҲҳ кё°к°„м—җ л§һм¶° н•„мҡ”н•ң м„ңлҘҳлҘј мӨҖл№„н•ҳкі , мҳЁлқјмқёмңјлЎң мӢ мІӯн•ҙм•ј н•©лӢҲлӢӨ. мӢ мІӯ нӣ„м—җлҠ” мІӯм•Ҫ лӢ№мІЁ л°ңн‘ңлҘј кё°лӢӨлҰ¬л©ҙ лҗ©лӢҲлӢӨ. мІӯм•ҪмӢ мІӯ мӢң мЈјмқҳн•  м җмқҖ лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.

мӢ мІӯ л°©лІ• м„ёл¶ҖмӮ¬н•ӯ
#

  1. нҡҢмӣҗк°Җмһ… л°Ҹ лЎңк·ёмқё: мІӯм•ҪнҷҲм—җ нҡҢмӣҗк°Җмһ… нӣ„ лЎңк·ёмқён•©лӢҲлӢӨ.
  2. мІӯм•ҪмӢ мІӯ л©”лүҙ м„ нғқ: мІӯм•ҪмӢ мІӯ л©”лүҙлҘј нҶөн•ҙ мӣҗн•ҳлҠ” кіөкі лҘј м„ нғқн•©лӢҲлӢӨ.
  3. мӢ мІӯм„ң мһ‘м„ұ л°Ҹ м ңм¶ң: кё°мһ…мӮ¬н•ӯмқ„ м •нҷ•нһҲ мһ…л Ҙн•ҳкі  м„ңлҘҳлҘј мІЁл¶Җн•ҳм—¬ м ңм¶ңн•©лӢҲлӢӨ.
  4. кІ°кіј нҷ•мқё: мІӯм•Ҫ кІ°кіјлҠ” мІӯм•ҪнҷҲм—җм„ң нҷ•мқён•  мҲҳ мһҲмңјл©°, лӢ№мІЁ нӣ„ кі„м•Ҫ м Ҳм°Ёк°Җ н•„мҡ”н•©лӢҲлӢӨ.

мІӯм•Ҫ лӢ№мІЁ нҷ•лҘ  лҶ’мқҙлҠ” нҢҒ
#

мІӯм•Ҫм—җ лӢ№мІЁлҗҳкё° мң„н•ҙм„ңлҠ” лӘҮ к°Җм§Җ м „лһөмқ„ м„ёмҡ°лҠ” кІғмқҙ мўӢмҠөлӢҲлӢӨ.

  1. мІӯм•ҪнҶөмһҘ к°Җмһ… кё°к°„ лҠҳлҰ¬кё°: мІӯм•ҪнҶөмһҘмқ„ мҳӨлһҳ мң м§Җн• мҲҳлЎқ к°Җм җмқҙ мҳ¬лқјк°‘лӢҲлӢӨ.
  2. мҶҢл“қ л°Ҹ мһҗмӮ° кҙҖлҰ¬: мҶҢл“қкіј мһҗмӮ°мқ„ кҙҖлҰ¬н•ҳм—¬ мЎ°кұҙмқ„ м¶©мЎұн•ҳлҠ” кІғмқҙ мӨ‘мҡ”н•©лӢҲлӢӨ.
  3. м—¬лҹ¬ кіөкі м—җ мӢ мІӯн•ҳкё°: к°ҖлҠҘн•ң л§ҺмқҖ кіөкі м—җ мӢ мІӯн•ҳм—¬ лӢ№мІЁ нҷ•лҘ мқ„ лҶ’мқҙлҠ” кІғмқҙ мўӢмҠөлӢҲлӢӨ.

н•өмӢ¬ мҡ”м•Ҫ
#

2026л…„ 03мӣ”м—җлҠ” лӢӨм–‘н•ң көӯлҜјмһ„лҢҖмЈјнғқ мІӯм•Ҫмқҙ м§„н–үлҗ©лӢҲлӢӨ. мІӯм•Ҫ мһҗкІ© л°Ҹ н•„мҡ”н•ң м„ңлҘҳлҘј лҜёлҰ¬ мӨҖл№„н•ҳм—¬ мӢ мІӯн•ҳм„ёмҡ”. мІӯм•ҪнҷҲм—җм„ң к°„нҺён•ҳкІҢ мӢ мІӯн•  мҲҳ мһҲмңјл©°, лӢ№мІЁ нҷ•лҘ мқ„ лҶ’мқҙкё° мң„н•ҙ м „лһөм ҒмңјлЎң м ‘к·јн•ҳлҠ” кІғмқҙ мӨ‘мҡ”н•©лӢҲлӢӨ.
